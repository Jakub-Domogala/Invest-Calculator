# Investment Calculator

**[Live site](https://jakub-domogala.github.io/Invest-Calculator/)**

An interactive calculator for projecting investment growth. Set your initial investment, monthly contributions, expected return, and inflation, and see your balance grow over time on a live chart — every field recalculates instantly, no submit button required.

## Tech stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (Base UI primitives)
- Recharts
- Number Flow
